# 🔢 Counter App

A simple and responsive Counter App built using **React.js**, **CSS**, and **Font Awesome**.
This app allows users to increase, decrease, and reset the counter with a clean and modern UI.

## 🚀 Features

* ➕ Increase counter
* ➖ Decrease counter
* 🔄 Reset counter
* 🎨 Stylish UI with gradient design
* 📱 Responsive layout
* ⚡ Fast and lightweight
* 🌙 Dark mode

## 🛠️ Technologies Used

* React.js
* JavaScript
* CSS
* Font Awesome
* HTML
